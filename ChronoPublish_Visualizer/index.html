Welcome to Publication Timeline Visualizer

This app allows you to search for articles or visualize your publication timeline from an Excel file. Here's how it works:

  1. Search for articles using the arXiv API or upload an Excel file containing your publication data.
  2. The app will generate a chart showing publications over time.
  3. Use the filters to explore data by author or year.
  4. View publication statistics below the chart.

Your Excel file should have these columns:

  • year: The publication year
  • title: The title of the publication
  • author: The author's name
  • date: The publication date
  • link: The link to the article
